# 🤖 RAG Chat API (.NET)

A **Retrieval-Augmented Generation (RAG)** API built with **ASP.NET Core** that combines Large Language Models (LLMs) with vector search to generate context-aware responses from custom documents.

## ✨ Features

- Document ingestion and indexing
- Text embedding generation
- Vector search
- RESTful API endpoints
- Scalable and modular architecture

## 🛠️ Tech Stack

- ASP.NET Core
- C#
- OpenAI Embeddings
- Qdrant Vector Database
- REST APIs

---

⭐ This project demonstrates how Retrieval-Augmented Generation (RAG) can be integrated into .NET applications to build intelligent, document-aware AI solutions.
